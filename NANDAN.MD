## 2. Architecture & Tech Stack (80-90 Seconds)
*"Let me quickly walk you through how CashGuardian is built under the hood.*

*Every user starts with JWT authentication — secure login with role-based access, so only the right people see the right data.*

*Once in, they land on the frontend — the dashboard, data visualizations, AI powered insights, and the ability to download reports.*

*All requests go through an API gateway, which handles rate limiting, auth verification, request validation, and response handling. This is what keeps the system secure and stable.*

*From there, the intelligence core takes over. A LangGraph agent receives the request, passes it to the Query Agent, which maps the user's intent into structured actions via the Intent Mapper. Before anything is sent to the LLM, our deterministic service layer runs the numbers. The AI then takes those numbers and weaves them into a professional narrative.*

*Finally, everything is logged. We have an audit trail for every query, every login, and every system action, ensuring full accountability for financial decision-making."*
